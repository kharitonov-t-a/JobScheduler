Установка (Вариант 1)
Распокуйте архив в корень.
Запустите index.php.
Следуйте инструкциям (заполняйте поля и нажимайте далее).

Установка (Вариант 2)
Распокуйте архив в корень.
Перенесите все из папки install в корень с заменой совпадающих файлов.
Переименуйте файл config-exemple.php в config.php
В config.php введите следующие данные - $HOST, $USER, $PASS, $DB.
Создайте базу данных $DB.
Импортируйте в нее test.sql из корня.
Запустите index.php.
В програме создайте пользвателя с логином admin (это будет суперпользователь).
